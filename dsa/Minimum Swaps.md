
Given an array of integers **A** and an integer **B**, find an return the minimum number of swaps required to bring all the elements less than or equal to **B** together. 

**Note:** It is possible to swap any two elements, not necessarily consecutive.

**Problem Constraints:**

1 <= length of the array <= 10<sup>5</sup>  
-10<sup>9</sup> <= A\[i\], B <= 10<sup>9</sup>
