

### Interpolation is done using Template Literal String 

Syntax : `${...}`

```javascript
let myName = 'Swar';

console.log(`Hi! My name is ${myName}`);
```



#### `fetch()`

The `fetch()` method returns a `Promise`, which is a placeholder object that will either be fulfilled if your request is successful, or rejected if your request is unsuccessful. 

If the `Promise` is fulfilled, it resolves to `Response` object, and you can use the `then` method to access the `fetch()` method: 

```js
fetch("https://example.org")
.then((res) => res)
```
